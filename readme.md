# RISC_V_MASTERY

A structured documentation repository for my RISC-V learning journey, workshop implementations, microarchitecture exploration, and hands-on CPU design projects.

---

## About This Repository

This repository documents my complete learning journey through:
- RISC-V architecture
- CPU microarchitecture
- digital logic implementation
- pipelined processor concepts
- RTL design
- verification
- and open-source hardware development

The repository is being maintained as part of my workshop and project-based learning process with a strong focus on:
- implementation,
- debugging,
- documentation,
- and practical understanding.

This repository is not intended to be just a collection of notes or copied labs.

The goal is to create:
- a well-structured engineering portfolio,
- a transparent learning journey,
- and a useful reference for other learners exploring RISC-V and processor design.

## Documentation Style

Each module or project will include:

### Overview
Brief explanation of the concept or objective.

### Theory Notes
Important architecture and implementation concepts.

### RTL / Code
Implemented source code and logic.

### Simulation & Outputs
Screenshots, logs, waveforms, and execution results.

### Debugging Notes
Issues faced during implementation and how they were resolved.

### Learnings
Important takeaways from the module or project.
